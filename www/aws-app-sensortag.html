Telia IoT Texas Instrument SensorTag IoT Demo

Status: Click the activate button on the SensorTag and tap the Start button to begin

Instructions:

This app demonstrates how to upload data from a Texas Instruments Bluetooth Smart SensorTag to the Telia IoT service.

Edit the source code of file aws-config.js to use your Telia IoT credentials. See the README file for further information.

When ready to go, activate the SensorTag and press the Start button to connect and start uploading sensor data.
